# mytraceroute

***mytraceroute*** is a simple implementation of the *traceroute* application. It identifies the number of hops to given domain. 

### Parameters

* Number of retries if ICMP time limit exceeded : 3
* Max. TTL : 16 

### Usage

```bash
make mytraceroute
sudo ./mytraceroute <domain-name>
```
